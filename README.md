# CAD to shapely

convert a CAD file to [shapely](https://pypi.org/project/Shapely/) geometry for downstream processing.
Currently supports, to varying degrees:

- DXF using [ezdxf](https://pypi.org/project/ezdxf/)
- (IGES to follow)
- (SVG to follow)
